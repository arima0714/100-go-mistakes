# No.17: 8進数リテラルで混乱を招く

0から始まる数値を入力すると8進数として扱われるので注意。

* 0b, 0B ではじめると2進数
* 0, 0o ではじめると8進数
* 0x, 0X ではじめると16進数
* i で終わらせると虚数

_ で数値を区切ることもできる。見やすい。

```
package main

func main() {
	sum := 100 + 010
	print(sum) // 108 と出力される

	sum = 100 + 0o10
	print(sum) // 108 と出力される

	bin := 0b10
	print(bin) // 2 と出力される

	hex := 0x10
	print(hex) // 16 と出力される

	ima := 3i
	print(ima) // 3i と出力される

    dec := 1_000_000_000
    print(dec) // 1000000000 と出力される
}

```
